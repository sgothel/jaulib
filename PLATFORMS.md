# Jau Support Library (C++, Java, ...)

[Original main document location](https://jausoft.com/cgit/jaulib.git/about/).

## Supported Platforms
Minimum language requirements
- C++17
- Java 11 (optional)

The following **platforms** are tested and hence supported

**Debian 12 Bookworm (GNU/Linux)**
- compiler
  - gcc 11.2.0
  - clang 13.0.1
  - openjdk 17
- architectures
  - amd64 (validated, Generic)

**Debian 11 Bullseye (GNU/Linux)**
- compiler
  - gcc 10.2.1
  - clang 11.0.1
  - openjdk 17
- architectures
  - amd64 (validated, Generic)
  - arm64 (validated, Raspberry Pi 3+ and 4)
  - arm32 (validated, Raspberry Pi 3+ and 4)

**Debian 10 Buster (GNU/Linux)**
- *deprecated*
- compiler
  - gcc 8.3.0
  - openjdk 11
- architectures
  - amd64 (validated, Generic)
  - arm64 (validated, Raspberry Pi 3+ and 4)
  - arm32 (validated, Raspberry Pi 3+ and 4)
- potential issues with *capsh*, see below.

**Ubuntu 22.04 (GNU/Linux)**
- compiler
  - gcc 11.2.0
  - clang 14.0.0
  - openjdk 17
- architectures
  - amd64 (validated, Generic)

**Ubuntu 20.04 (GNU/Linux)**
- compiler
  - gcc 9.4.0
  - clang 10.0.0
  - openjdk 17
- architectures
  - amd64 (validated, Generic)

**Ubuntu 18.04 (GNU/Linux)**
- *deprecated*
- compiler
  - gcc 8.3
  - openjdk 11
- architectures
  - amd64 (validated, Generic)
- potential issues with *capsh*, see below.
